# Collab TODO List

Classic TODO list, but collab!

## Made with

- Vainila Javascript
- JQuery (Only for open/close modal)
- BootStrap 4

## Instructions

- First, install all dependencies:
```
npm install
```

- Test in localhost:

Build app.js
```
npm run build
```

Start the server
```
npm run start
```
